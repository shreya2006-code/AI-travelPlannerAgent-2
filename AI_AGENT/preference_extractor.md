The Preference Extractor should identify the following from every user message:

- Destination
- Budget
- Number of days
- Number of travelers
- Travel type (Solo, Couple, Family, Friends)
- Preferred categories (Nature, Heritage, Adventure, Beach, Wildlife, etc.)
- Excluded categories (Temple, Museum, Nightlife, etc.)
- Preferred hotel type (Budget, Mid-range, Luxury)
- Special preferences (Vegetarian, Kid-friendly, Senior-friendly, etc.)

If any information is missing, the AI should make reasonable recommendations or ask follow-up questions only when necessary.
