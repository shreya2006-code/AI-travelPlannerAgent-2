AI Itinerary Generation Flow

1. Extract user preferences.

2. Search Places dataset.

3. Retrieve matching hotels.

4. Retrieve matching attractions.

5. Select the most suitable hotel.

6. Select attractions based on user preferences.

7. Arrange attractions into a day-wise itinerary.

8. Pass the structured itinerary to the LLM.

9. The LLM converts it into a conversational response.

10. Save the itinerary in Trip Context.
